# Lavina Arsya - Personal Portfolio

A modern, responsive personal portfolio website built to showcase my projects, skills, and experience as a Fullstack Developer and Machine Learning Engineer.

## 🚀 Built With

- **[Next.js 15](https://nextjs.org/)** - The React Framework for the Web
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid UI development
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animation library for React
- **[Lucide React](https://lucide.dev/)** - Beautiful & consistent icon toolkit
- **[Formspree](https://formspree.io/)** - Functional contact form handling

## ✨ Features

- **Modern UI/UX**: Glassmorphism design with smooth animations.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop.
- **Interactive Tech Stack**: Auto-scrolling marquee showcasing Fullstack & AI/ML skills.
- **Project Showcase**: Detailed cards with links to GitHub and Demos.
- **Working Contact Form**: Integrated with Formspree for direct email messaging.

## 🛠️ Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Arsyacoo/cv-lavina.git
    cd cv-lavina
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## 📝 Configuration

To make the contact form work, update `src/components/Contact.tsx` with your own Formspree ID:

```typescript
const FORMSPREE_ID = "YOUR_ID_HERE";
```
